About This Project

This project uses machine learning to predict whether a person is affected by Parkinson's disease based on their voice sample.

Parkinson's disease is a neurodegenerative disorder that affects movement. Detecting it early can significantly improve the quality of life for those affected.

By analyzing voice samples, we aim to create a tool that helps in early detection, contributing to better healthcare solutions.
